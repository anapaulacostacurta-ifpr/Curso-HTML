Exercicio 03: Criar arquivo index.html sem tags HTML

Nesta atividade, você irá criar um arquivo HTML simples com texto puro (sem tags HTML), contendo três mensagens em idiomas diferentes. Depois, você irá visualizar o resultado no navegador para observar o comportamento da codificação.

📄 Conteúdo do arquivo index.html:

    Olá, mundo!
    ¡Hola, mundo!
    こんにちは、世界!

🛠️ Passo a passo para criar o arquivo no GitHub

  1. Acesse o repositório no GitHub: https://github.com/seu-usuario/laboratorio-front-end
  2. Dentro do repositório, clique em "Add file" (Adicionar arquivo) e escolha "Create new file" (Criar novo arquivo).
  3. Na caixa de nome do arquivo, digite: atividade-01-sem-tags/index.html. Isso criará a pasta atividade-01-sem-tags e o arquivo index.html dentro dela.
  4. Copie e cole o conteúdo do arquivo (texto puro) no editor:
    5.     Olá, mundo!
    ¡Hola, mundo!
    こんにちは、世界!
  5. Role a página para baixo, preencha a mensagem de commit, por exemplo: Cria arquivo index.html com mensagens em três idiomas
  6. Clique no botão verde "Commit new file" para salvar o arquivo no repositório.

🌐 Como acessar sua página no navegador

Após publicar no GitHub Pages, a URL para acessar a página será:

    https://seu-usuario.github.io/laboratorio-front-end/atividade-01-sem-tags/

Onde:

  • seu-usuario deve ser substituído pelo seu nome de usuário no GitHub.
  • laboratorio-front-end é o nome do repositório.
  • atividade-01-sem-tags é a pasta onde o arquivo index.html está.
💡 Dica: O GitHub Pages usa o arquivo index.html como página inicial para cada pasta. Caso a página não apareça, aguarde alguns minutos para o GitHub atualizar o site.

✅ Objetivo da atividade

O objetivo é que você perceba como o navegador interpreta caracteres especiais e diferentes alfabetos, mesmo quando o arquivo não possui marcações HTML.
