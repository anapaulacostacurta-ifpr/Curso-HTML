Sistema de Pontos: IFPRPoints

Para incentivar a prática e a conclusão dos exercícios, cada atividade correta vale uma quantidade de IFPRPoints, nossos pontos oficiais do laboratório de Front-end!

👉 O que são IFPRPoints?

IFPRPoints são pontos virtuais que você acumula ao concluir exercícios com qualidade. Você pode trocar esses pontos por vantagens especiais com a professora.

🏆 Como funciona?

  • Os pontos são acumulados ao completar os exercícios e apresentar os resultados no GitHub Pages.
  • Verificamos se a estrutura do código está correta, organizada (indentação adequada), e se os arquivos estão no GitHub com o link do GitHub Pages funcionando perfeitamente.
  • Ao final de cada mês, será realizado ao ranking dos estudantes com mais pontos!

🚀 Comece a acumular seus IFPRPoints agora!

Para cada exercício, certifique-se de:

  • Estruturar o código corretamente
  • Organizar o código com recuo (indentação) adequado
  • Publicar os arquivos no GitHub
  • Garantir que o link do GitHub Pages funcione perfeitamente
