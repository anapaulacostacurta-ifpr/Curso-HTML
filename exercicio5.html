Exercício: Pesquisa, Criação e Revisão de Página HTML

Passo 1 — Escolha do tema

- Acesse a seção HTML do W3Schools: https://www.w3schools.com/html/ - Escolha um tópico novo para você (exemplo: Listas, Tabelas, Formulários, Elementos Semânticos, etc.). - Leia e estude o conteúdo da página. - Você pode usar o ChatGPT para tirar dúvidas, pedir explicações alternativas ou exemplos para entender melhor o tema. Por exemplo: “ChatGPT, explique para mim o que são listas em HTML com um exemplo simples”.

Passo 2 — Montagem da página

- Crie uma página HTML (index.html) que contenha:

  • Um título com o nome do tema escolhido (exemplo: "Listas em HTML").
  • Um parágrafo que explique com suas próprias palavras o que é o tema.
  • Um exemplo simples em HTML que ilustre o tema (pode ser exibido em <pre>, ou renderizado na própria página).
  • Uma imagem relacionada, se quiser.
  • Pelo menos 2 links relacionados:
    • Link para a página do W3Schools sobre o tema,
    • Um outro link complementar (exemplo: vídeo, artigo).

Importante: Use apenas os elementos HTML que você já aprendeu até o momento (títulos, parágrafos, imagens, links, listas, etc.). Não use CSS ou estilização ainda.

Passo 3 — Troca e revisão

- Troque a página com outro aluno.
- Use um checklist para revisar:

  • A página abre corretamente e o conteúdo está claro?
  • A explicação está coerente e com suas próprias palavras?
  • O exemplo de código funciona?
  • Os links estão corretos?
  • O código HTML está organizado e bem indentado?
  • Sugestões de melhoria.

Passo 4 — Feedback

Retorne o feedback para o colega.

Dicas para usar o ChatGPT com responsabilidade

  • Use o ChatGPT para entender o conteúdo, tirar dúvidas e pedir exemplos.
  • Não copie respostas diretamente, tente reformular com suas próprias palavras para aprender melhor.
  • Se pedir código, entenda cada linha antes de usar.
  • Use o ChatGPT para revisar seu texto, mas também faça sua própria reflexão crítica.
