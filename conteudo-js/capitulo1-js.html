Capítulo 1 – Introdução ao JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas HTML. Com JavaScript, você pode responder a cliques de botões, validar formulários, alterar o conteúdo da página e muito mais.

Onde usar o JavaScript?

  • Dentro da própria página HTML (com a tag <script>)
  • Em arquivos externos (com extensão .js)

Primeiro exemplo

Vamos mostrar uma mensagem ao usuário:


                    <script>
                    alert("Bem-vindo à minha página!");
                    </script>

Inserindo JavaScript no HTML

Você pode colocar o código JavaScript diretamente no final do <body>, assim: 


                    <body>
                    <h1>Olá!</h1>
                    <script>
                        console.log("Mensagem no console.");
                    </script>
                    </body>

Comandos básicos

  • alert("texto") → mostra uma janela pop-up
  • console.log("texto") → escreve no console do navegador (F12)
  • document.getElementById() → acessa elementos HTML pelo id

Exemplo interativo com botão

HTML:


                    <b  utton onclick="mostrarAlerta()">Clique aqui</button>

JavaScript:


                    <script>
                    function mostrarAlerta() {
                        alert("Você clicou no botão!");
                    }
                    </script>

Diferente de Java

Apesar do nome parecido, JavaScript e Java são linguagens diferentes. O JavaScript é usado principalmente em páginas web.

Resumo

  • JavaScript torna páginas web interativas e dinâmicas.
  • Você pode escrever código dentro da tag <script>.
  • Comandos como alert() e console.log() ajudam a testar o código.
  • Eventos como cliques podem ser usados para disparar ações.
