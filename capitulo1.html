Introdução ao HTML – Criando a Estrutura de um Site

O que é HTML?

Se você quer aprender a criar sites, precisa começar com HTML! Mas o que exatamente é isso?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem que usamos para estruturar o conteúdo das páginas web. Ao contrário de linguagens de programação, o HTML não executa ações — ele apenas organiza e apresenta informações.

Elementos e Tags: Os Blocos de Construção da Web

HTML é construído com elementos, e cada elemento é representado por tags.

As tags servem para “rotular” diferentes partes do conteúdo, como:

  • Títulos: <h1>, <h2>
  • Parágrafos: <p>
  • Listas: <ul>, <ol>
  • Tabelas: <table>
  • Links: <a>

Essas tags não aparecem visualmente na página, mas dizem ao navegador como mostrar cada parte do conteúdo.

Vamos Começar na Prática

Veja abaixo um exemplo simples de código HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Minha Primeira Página</title>
</head>
<body>
  <h1>Bem-vindo!</h1>
  <p>Este é o meu primeiro parágrafo em HTML.</p>
</body>
</html>

Explicação:

Elemento Função
<!DOCTYPE html>Declara que o documento usa HTML5
<html>Elemento raiz da página
<head>Contém informações invisíveis, como título e codificação
<title>Define o título exibido na aba do navegador
<body>Contém o conteúdo visível da página
<h1>Título principal da página
<p>Parágrafo de texto

Como as Tags Funcionam

A maioria das tags HTML vêm em pares:

Uma tag de abertura, por exemplo: <p>
Uma tag de fechamento: </p>

O conteúdo fica entre elas:

<p>Esse é um parágrafo.</p>

Algumas tags são autoencerradas, como <br>, e não precisam de uma tag de fechamento.

O Papel dos Navegadores

O HTML por si só não cria um site visualmente bonito. Quem transforma o código em uma página visível é o navegador.

Navegadores como Chrome, Firefox, Edge ou Safari interpretam o HTML e exibem o conteúdo corretamente na tela.
