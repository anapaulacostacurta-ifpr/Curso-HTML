💰 Sistema de Pontos: IFPRPoints

Para incentivar a prática e a conclusão dos exercícios, cada atividade correta vale uma quantidade de pontos convertidos na moeda virtual IFPRPoints.

🪙 IFPRPoint é ponto oficial do nosso laboratório de Front-end! Acumule e troque seus IFPR por vantagens com a professora.

🏆 Regras

  1. Os pontos são acumulados ao concluir os exercícios e apresentar o resultado corretamente no GitHub Pages.
  2. As revisões de código serão feitas semanalmente para validar os pontos.
  3. Prêmios especiais serão oferecidos aos estudantes com mais pontos no final de cada mês.

🚀 Comece a acumular seus IFPRPoints agora!

Para cada exercício, verifique se a estrutura está correta, se está organizado com espaço ou recuo em relação à margem (indentados), se os arquivos estão no GitHub e o link do GitHub Pages funciona perfeitamente.
